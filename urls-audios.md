## Como usar

1. Após fazer o push deste repositório para o GitHub, substitua `Leonardotrentini` pelo seu nome de usuário real
2. Copie a URL completa do áudio desejado
3. Use essa URL no Google Studio para vincular o áudio ao módulo correspondente

Há um arquivo duplicado: `MÓDULO 4 – 50 TIPS PARA UNA MEMORIA IMPARABLE(1)(1).MP3`. Certifique-se de usar apenas uma das versões no Google Studio.
