# AudioSpulse - Repositório de Áudios

Este repositório contém arquivos de áudio hospedados para uso com Google Studio.

## Estrutura

Os áudios estão organizados na pasta `audios/`.

## Como usar

Você pode acessar os áudios diretamente através das URLs do GitHub Raw:

```
https://raw.githubusercontent.com/[seu-usuario]/audiospulse/main/audios/[nome-do-arquivo]
```
