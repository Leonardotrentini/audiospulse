# AudioSpulse - Google Studio App

Este repositório contém a aplicação do Google Studio (AI Studio) e os arquivos de áudio hospedados para uso com os módulos.

View your app in AI Studio: https://ai.studio/apps/drive/1fWo6CTyxWLTf9n2NJixcIcigaUn4__R0

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   ```bash
   npm run dev
   ```

## Arquivos de Áudio Hospedados

Este repositório também hospeda arquivos de áudio para uso com Google Studio.

### Estrutura

Os áudios estão organizados na pasta `audios/`.

### Como usar os áudios

Você pode acessar os áudios diretamente através das URLs do GitHub Raw:

```
https://raw.githubusercontent.com/Leonardotrentini/audiospulse/main/audios/[nome-do-arquivo]
```
